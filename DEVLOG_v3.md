## 📅 03.03.2026 (вторник)

### ✅ Сделано
- **Этап 1: Архитектура Задач (Время, Циклы, Подзадачи)**
  - Создан новый компонент `TimePicker` для выбора времени (HH:MM) с поддержкой Web (react-datepicker) и Native (DateTimePicker).
  - В модалку создания/редактирования задачи добавлен блок "Дополнительные настройки" (раскрывашка).
  - В дополнительные настройки интегрирован выбор времени и выбор цикличности (каждый день, неделю, месяц).
  - В карточке задачи теперь отображается установленное время и значок 🔄 для цикличных задач.
  - Убрано редактирование по долгому тапу (освобождено для будущего drag-n-drop).
  - На карточку задачи (в правый верхний угол) добавлена явная кнопка ✏️ "Редактировать".
  - Оптимизирована отрисовка свайпа (убран красный контур, уменьшена чувствительность свайпа влево для удаления).
  - Добавлена кнопка "Остановить повторения" при редактировании цикличной задачи.
  - Логика подзадач: теперь мы ожидаем `subtasks_count` от бэкенда при загрузке списка задач (или показываем длину массива загруженных подзадач), чтобы значок подзадач 📋 отображался сразу, не дожидаясь раскрытия аккордеона.

### 🔄 В процессе
- Ожидание бэкенд-реализации (на Beget) создания копии задачи при выполнении (если `isrecurring = 1`).
- Переход к Этапу 2 (Категории и папки).

### 📝 Заметки
- Для работы `TimePicker` в мобильной версии используется `@react-native-community/datetimepicker` с модом `time` и `is24Hour={true}`.
