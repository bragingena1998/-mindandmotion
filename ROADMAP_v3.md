## 🎯 Этап 1: Архитектура Задач (Время, Циклы, Подзадачи)
- [ ] **Добавление времени (Time):** Установка точного времени (напр. 14:00) через прокручивающееся колесо (24h). Настройка скрыта в "раскрывашке" (коллапс) при создании.
- [ ] **Цикличность:** Повторение задачи (раз в неделю, месяц, по выбранным дням). При выполнении текущей — создается копия на следующий период.
- [ ] **Управление циклами:** Возможность отвязать/выключить повторение у будущих задач без удаления истории предыдущих.
- [ ] **Подзадачи:** Мгновенная подгрузка списка подзадач (значок виден сразу, без переоткрытия).
- [ ] **UI Управление:** 
  - Вернуть кнопку "Редактировать" в правый верхний угол карточки (разгрузить жесты).
  - Свайп для удаления остается (убрать странный красный квадратный контур вокруг кнопки удалить).

## 📁 Этап 2: Категории (Папки) как Фильтры
- [ ] **Создание папок:** Сущность "Папка" (Название + Иконка/Стикер - пока стандартные эмодзи).
- [ ] **Отображение (Фильтры):** Горизонтальный список папок сверху (Общие, Работа, Личное). Клик работает как визуальный фильтр списка и архива, без проваливания внутрь экранов папок.
- [ ] **Быстрый перенос:** Перетаскивание (drag-and-drop) задачи долгим тапом в папку сверху.
- [ ] **UI Задачи:** В общем списке на карточке задачи видна иконка ее папки.
- [ ] **UI Создания:** Выбор папки скрыт в ненавязчивом раскрывающемся меню при создании задачи.

## 🍅 Этап 3: Концентрат-сессия (Focus Mode)
- [ ] Удобный UI-ввод таймера концентрации (барабан).
- [ ] Кнопка запуска "Концентрат-сессии" внутри карточки задачи.
- [ ] Сессия работает в фоне: по истечении времени приходит Push-уведомление (с возможностью прервать досрочно).

## ⚡ Этап 4: Привычки (Логика и Багфиксы)
- [ ] **Багфикс:** Исправить неработающее удаление привычек.
- [ ] **Логика дат:** Убрать обязательный выбор диапазона дат, если привычка создается "на месяц" или бессрочно.

## 📅 Этап 5: Календарь (Глобальное обновление)
- [ ] **Будущие планы:** Отображение ВСЕХ запланированных (будущих) задач и привычек в сетке календаря. Главная суть — видеть объем работы на будущее.
- [ ] **Индикация статуса:** Разделение визуально: зачеркнутые (выполнены) / активные (ждут).
- [ ] **Фокус дня:** По клику на день — показ задач + привычек, которые должны быть выполнены именно *сегодня* по расписанию.
- [ ] **Дни Рождения:** Барабан выбора даты (как при регистрации). Вывод информации: "Ивану исполняется 28 лет" + уведомление.

## 🔔 Этап 6: Уведомления (Notifications)
- [ ] **Тонкая настройка:** Экран выбора уведомлений (утро, вечер, дни рождения, таймеры).
- [ ] **Напоминания до дедлайна:** За 10 минут до задачи (если у задачи установлено время).
- [ ] **Уборка:** Удалить тестовую кнопку уведомлений.

## 🎓 Этап 7: Онбординг и Демо-данные
- [ ] **Интерактивный туториал:** Подробная инструкция (overlay) при первом запуске на каждом экране (подсветка свайпов, скрытых функций). Обязательно покрыть весь функционал привычек (часы, количество, дни недели, месяц).
- [ ] **Кнопка "Справка":** Едва заметная иконка "?" на экранах для перезапуска туториала.
- [ ] **Шаблонное заполнение:** Автоматическое создание демо-привычек и демо-задач при новой регистрации, чтобы наглядно показать весь функционал и состояния (текущие, будущие, просроченные).

## ✅ Этап 1: Архитектура задач
> Приоритет: 🔴 Высокий

- [ ] **Время задачи** — барабан выбора времени (HH:MM, 24h), скрыт в раскрывашке «Дополнительно» при создании/редактировании
- [ ] **Цикличность** — повторение: каждый день / каждую неделю / каждый месяц / конкретные дни недели. Скрыто в раскрывашке «Дополнительно»
- [ ] **Логика сервера** — при выполнении цикличной задачи (`isrecurring=1`) сервер автоматически создаёт копию на следующий период
- [ ] **Остановить повторения** — кнопка в модалке редактирования цикличной задачи (отвязывает будущие без удаления истории)
- [ ] **Подзадачи** — значок 📋 на карточке задачи отображается сразу через `subtasks_count` с бэкенда (без открытия/закрытия аккордеона)
- [ ] **Редактирование** — кнопка ✏️ в правом верхнем углу карточки задачи
- [ ] **Удаление** — перенести кнопку «Удалить» внутрь модалки редактирования (осознанное действие, защита от случайного удаления)
- [ ] **Свайп влево** — запуск Концентрат-сессии (не удаление)
- [ ] **Концентрат-сессия** — барабан выбора времени сессии → запуск таймера → push-уведомление по окончании с возможностью прервать досрочно
- [ ] **Багфикс** — убрать красный квадратный контур вокруг кнопки удаления

  id INT AUTO_INCREMENT PRIMARY KEY,
