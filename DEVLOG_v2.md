## 📅 18.02.2026 (среда)

### ✅ Сделано
- **Развернут Dev Client:** Успешно собрали первый Development Build через EAS Cloud.
  - Версия: `development`
  - Платформа: Android
  - Метод сборки: Облачная (EAS Build)
- **Устранены проблемы окружения:**
  - Локальная сборка `run:android` на Windows признана нестабильной (конфликты путей Gradle/Kotlin).
  - Перешли на гарантированно рабочий вариант через EAS.
- **Интеграция:** Приложение запускается на телефоне, Hot Reload работает, логи выводятся в терминал.

### 🔄 В процессе
- Исправление ошибки рендеринга текста (`Text strings must be rendered within a <Text> component`) в `TasksScreen`.

### ⚠️ Проблемы / Вопросы
- **РЕШЕНО:** `run:android` падал с ошибкой плагинов. Решение — использовать EAS Build (`eas build --profile development --platform android`).
- **РЕШЕНО:** `crypto` error в `axios`. Решение — даунгрейд `axios` до 1.6.8.

### 📝 Заметки
- **Важно:** Для разработки всегда запускать `npx expo start --dev-client`.
- Не пытаться чинить локальный Android-билд на Windows, это трата времени.
